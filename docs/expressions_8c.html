PARSERFUNC ( atom  )

Tries to accept a floating point constant.

if it couldn't find one it tries an Integer constant, if it couldn't find one it tries a String constant, if it couldnt find one etc... and returns this.

TODO: All other constants (string, list etc) NOTE: The order of float first int last is on purpose as the float function will error and return much earlier. The int function first checks for all different bases.

rule: atom = floatconst | intconst
