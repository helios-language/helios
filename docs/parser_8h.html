void parser_free ( Parser_t *  parser)

Parser destructor.

Destroys a parser object together with it's errorstack and string of accepted characters.

WARNING: this function will also delete the parser's errorstack. This may not be desired when freeing copies of a parser made with the parser_copy function as it will not copy the errorstack, only the pointer to the errorstack. Deleting this stack might/will crash the parser on subsequent accesses of the errorstack. Think about using the parser_free_simple function.
